# Webbiecorn SaaS Template

Standaard basisrepo voor alle Webbiecorn tools/SaaS/webapps.
Doel: sneller shippen, minder chaos, consistente kwaliteit.

## Wat zit erin?
- Docs: PRD / ADR / Runbook
- Issue templates (Idea / Feature / Bug)
- PR template checklist
- CI workflow (veilig: draait pas echt als er een package.json is)

## Workflow
1. Maak Issue
2. Maak branch
3. Open PR + checklist
4. Checks groen → merge
5. Release + changelog

## Mappen
- `docs/prd/` – requirements
- `docs/adr/` – belangrijke beslissingen
- `docs/runbook/` – ops & onderhoud
